# Features

One file per shipped/in-progress feature (use [[_templates/feature]]):
`docs/features/<feature-slug>.md`. Captures what it does, why it fits (story /
companion / horror / replay filter), how it's wired, and its decision/bug backlinks.
Mechanics-level design lives in `docs/mechanics/` and `docs/design/`; this folder is
the per-feature record that ties design → decisions → code.
